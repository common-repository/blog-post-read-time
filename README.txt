A plugin that shows the amount of time needed to finish reading a post.

== Description ==

Blog Post Read Time is a plugin that shows the average reading time needed to read a blog post. Ideally, it is believed that on average an adult can read 265 words in a minute. This plugin uses that information to estimate the amount of time needed to finish reading a blog post.  

NOTE: The amount of time shown does not cater for amount of time taken to view images if any is attached to the post.


== Installation ==

1. Upload `blog-post-read-time.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
